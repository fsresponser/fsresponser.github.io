大事件Ⅱ：论两种思维方式

2020-08-21 | 浏览

今天大厦闸机莫名关闭了，等待进入大厦办公的人们都在等待维修人员到场，门卫也在给维修部积极联系。正在大家万分急切之时，有一个人试图破机而入。结果：闸机并没有被关闭锁上，只是处于闭合状态而已，只要稍稍轻推就可通过。

人作为社会的产物，时刻受到社会的驯化。受到怎样的驯化呢，法律，规则，道德，风俗etc各有驯化条例。而驯化是一个温水煮青蛙的过程，这让我们在不知不觉中失去认识，认为事物之所以是这样是理所当然的。

  • 这会让我们固步自封，失去对问题进一步思考认识的能力（思）， 进而失去对未知领域(未知边界)进一步探索的机会（行）。就闸机事件，正确的反应应如是：

社会化思维：闸机关闭了 ——> 闸机在上班高峰时关闭了 ——> 闸机关闭的合理性 ——>结论：闸机在上班高峰期关闭不合理
社会化处理：不合理 ——> 由果导因 ——> 闸机不是我们看到的样子 ——> 踹一踹闸机是不是真的闭关锁上了还是只是闭合状体。

社会化认知（由社会化思维&社会化行动构成的反射弧）：大多数情况惯性思维是可以给出一件事情的合理判断，就闸机事件而言，我想所有人都会直觉性认为不合理。但是是什么阻断了反射弧的形成呢，是【行】。行在这次实践中作用突出（但不是说思的过程不重要，事实上非常重要，起到基石作用。只是这种思维我们大多数情况下是具备的，短板在行）。

延展： — 惯性思维只能处理简单事物，确不能处理复杂，创新，突破性事物（在创造性工作中，我们要尽量避免惯性思维，进取型思维的练习是有必要且重要的）。

工程化思维： 闸机关闭了 ——> {1: 闸机锁上了，2：闸机没锁上，只是处于闭合状态 } ——> { 1: 告知门卫联系维修部，尽快恢复正常，2:闭合状体, 轻踹即可 }

由此惊讶的发现，工程化的思维要远远简单于社会化思维，即工程化思维具备简单高效的特点。为什么会产生这种差异呢，我们发现工程化思维剥离了人的因素和环境因素（人对事物的判断，定义，推测、及周围环境对事物的影响），只专注于事物本身的状态（事物有多少种状态，当前是什么状态，产生当前状态的原因有哪些，以及原因对应的解决方法， 最终是实践方案。），快速迭代所有解决问题的所有方案。看似工程化的方案也是很复杂，但是它是有规可循，逻辑清晰，可操作性优良，实践简单。

  • 注意：我们得出任何的结论都是在背景，有条件的。通过两种思维的比较，我们发现工程化思维优势明显。—— 但是这个结论的背景是这个问题是闸机问题，它本身具有工程属性。所以我们在面对问题时，理性正确的思路应该是: 首先要认识到问题的属性，然后依附我们已经具备的条件， 采用相应的方法来处理问题。社会化思维和工程化思维就像我们解决问题的 两种工具（锤子），视具体问题分别挑选不同的工具才能事半功倍，高效解决问题。

社会化思维 — 1.思辨能力、 2.开采精神（甚至是破坏性开采）。
工程化思维 — 1.计算能力&逻辑能力、2. 实践能力。
